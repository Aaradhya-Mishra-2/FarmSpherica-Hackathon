# 🌱 Farmspherica

AI-Powered Hydroponic Farm Monitoring Platform

## Overview

Farmspherica is a smart agriculture platform designed to monitor hydroponically grown crops in real time. By combining computer vision, AI analytics, and live camera feeds, the platform helps farmers track crop health, growth, and environmental conditions through an intuitive dashboard.

This project was developed for a hackathon to demonstrate how artificial intelligence can make modern farming more efficient, accessible, and data-driven.

## Features

### 📊 Live Crop Dashboard

* Real-time monitoring of hydroponic crops
* Visual crop statistics and growth metrics
* Easy-to-understand analytics

### 🤖 AI-Powered Analysis

* Crop health assessment
* Growth pattern tracking
* Automated insight generation

### 📷 Camera Integration

* Connect farm cameras directly to the platform
* Continuous crop observation
* Live image-based monitoring

### 🌿 Smart Farming

* Hydroponic farming support
* Resource-efficient monitoring
* Scalable for farms of different sizes

## How It Works

1. Install cameras in the hydroponic farm.
2. Connect the cameras to Farmspherica.
3. AI processes incoming visual data.
4. Crop statistics and insights are generated automatically.
5. The dashboard updates in real time.

## Tech Stack

* Frontend: React / Next.js
* Styling: Tailwind CSS
* AI: Computer Vision & Machine Learning Models
* Backend: Node.js / Firebase / Supabase
* Deployment: Netlify

## Current Status

This is a prototype hackathon project. Some dashboard values currently use placeholder data for demonstration purposes. Future versions will utilize live camera feeds and real-time AI analytics.
